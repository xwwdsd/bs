## 1. 项目概述

### 1.1 项目背景
基于Steam平台的CS2（Counter-Strike 2）游戏饰品交易市场，为用户提供安全、便捷的饰品交易服务。

### 1.2 核心功能
- Steam账号绑定与库存同步
- 饰品求购（买家发布求购订单）
- 饰品供应（卖家出售饰品）
- 价格走势分析
- 钱包管理（充值、提现、交易结算）

### 1.3 用户角色
| 角色 | 描述 |
|------|------|
| 普通用户 | 注册并完成Steam绑定的用户，可进行交易 |
| VIP用户 | 满足特定条件的优质用户，享受更低手续费 |
| 管理员 | 平台运营人员，负责审核、风控、客服 |
| 超级管理员 | 系统最高权限，管理配置和敏感操作 |

#### 3.3.3 交易撮合机制
```
撮合流程:
1. 卖家发起供应/响应求购
2. 系统验证饰品有效性和所有权
3. 系统锁定交易饰品
4. 买家确认交易（自动/手动）
5. 系统发起Steam交易报价
6. 卖家确认发送报价
7. 买家确认接收报价
8. 系统检测交易完成状态
9. 资金结算到卖家钱包
10. 交易完成，更新统计数据
```

#### 3.3.4 交易状态流转
| 状态 | 说明 | 触发条件 |
|------|------|----------|
| 待确认 | 订单已创建，等待确认 | 买家提交订单 |
| 报价中 | Steam交易报价已创建 | 系统自动创建 |
| 待发货 | 等待卖家发送报价 | 卖家确认后 |
| 待收货 | 等待买家确认收货 | 卖家已发货 |
| 交易成功 | 交易完成 | 买家确认收货 |
| 交易取消 | 订单被取消 | 超时/主动取消 |
| 交易纠纷 | 交易出现问题 | 异常检测 |

## 4. 数据库设计

### 4.1 核心表结构

#### 4.1.1 用户相关表
```sql
-- 用户基础表
sys_user (
    id, username, email, password, avatar,
    steam_id, steam_trade_url, steam_api_key,
    status, user_level, is_real_name,
    created_at, updated_at
)

-- 用户实名认证表
user_verification (
    id, user_id, real_name, id_card_no,
    id_card_front, id_card_back, face_recognition,
    status, verified_at, created_at
)

-- 用户银行卡表
user_bank_card (
    id, user_id, bank_name, card_no,
    card_holder, is_default, created_at
)

-- 用户钱包表
user_wallet (
    id, user_id, balance, frozen_amount,
    total_recharge, total_withdraw,
    updated_at, created_at
)

-- 钱包流水表
wallet_transaction (
    id, user_id, type, amount, balance,
    order_no, remark, created_at
)
```

-- 用户库存表 (Steam库存快照)
user_inventory (

## 6. 安全与风控

### 6.1 安全措施

#### 6.1.1 认证授权
- JWT Token认证机制
- Token过期时间：Access Token 2小时，Refresh Token 7天
- 敏感操作需二次验证（交易确认、提现等）

#### 6.1.2 数据安全
- 用户密码采用BCrypt加密
- 敏感数据传输使用HTTPS
- 数据库敏感字段加密存储

#### 6.1.3 接口安全
- 接口防重放攻击（时间戳+随机数+签名）
- 接口限流（基于IP+用户）
- SQL注入防护（MyBatis参数化查询）
- XSS攻击防护

### 6.2 风控体系

#### 6.2.1 交易风控
| 风控规则 | 处理方式 |
|----------|----------|
| 价格异常（偏离市场价过大）| 人工审核 |
| 高频交易 | 触发验证码/限制交易 |
| 新用户大额交易 | 人工审核 |
| 交易对手异常 | 风险提示 |

#### 6.2.2 账号风控
| 风控规则 | 处理方式 |
|----------|----------|
| 异地登录 | 短信验证 |
| 密码错误多次 | 账号锁定15分钟 |
| 设备变更 | 邮件确认 |
| 疑似盗号 | 临时冻结 |

#### 6.2.3 资金风控
| 风控规则 | 处理方式 |
|----------|----------|
| 大额提现 | 人工审核 |
| 频繁提现 | 延迟处理 |
| 异常充值 | 冻结资金 |
